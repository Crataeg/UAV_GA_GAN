## 2. 高度变量与路径损耗的解析骨架

### 2.1 平均路径损耗重写

把代码里的 LoS/NLoS 平均项改写成更论文化的形式：

`(A1) L_{s,k} = FSPL(d_{s,k}) + eta_N - p_{s,k}^{LoS} * (eta_N - eta_L)`

其中：

`(A2) FSPL(d) = 20 log10(d) + 20 log10(f_c) + 20 log10(4 pi / c)`

### 2.2 高度导数

对 `h_k` 求导：

`(A3) dL_{s,k}/dh_k = dFSPL(d_{s,k})/dh_k - (eta_N - eta_L) * dp_{s,k}^{LoS}/dh_k`

若 `d_{s,k} = sqrt(r_{s,k}^2 + h_{s,k}^2)`，则

`(A4) dFSPL(d_{s,k})/dh_k = 20 / ln(10) * h_{s,k} / (r_{s,k}^2 + h_{s,k}^2)`

若 `p_{s,k}^{LoS} = 1 / (1 + C exp(-B(theta - C)))`，则

`(A5) dp_{s,k}^{LoS}/dtheta = B * p_{s,k}^{LoS} * (1 - p_{s,k}^{LoS})`

`(A6) dtheta/dh_k = 180 / pi * r_{s,k} / (r_{s,k}^2 + h_{s,k}^2)`

于是：

`(A7) dL_{s,k}/dh_k = 20 / ln(10) * h_{s,k} / (r_{s,k}^2 + h_{s,k}^2) - (eta_N - eta_L) * B * p_{s,k}^{LoS} * (1 - p_{s,k}^{LoS}) * 180 / pi * r_{s,k} / (r_{s,k}^2 + h_{s,k}^2)`

### 2.3 这条导数怎么用

这条式子非常适合拿来解释：

- 为什么高度不是越高越好
- 为什么某些差场景会出现在中高空而不是贴地
- 为什么不同 `LoS / NLoS` 参数下最坏高度会迁移

## 3. 聚合干扰的矩匹配推导

### 3.1 单个干扰项

定义：

`(B1) I_{i,k} = barOmega_{i,k} * G_{i,k}`

`(B2) G_{i,k} ~ Gamma(m_i, 1 / m_i)`

因此：

`(B3) E[I_{i,k}] = barOmega_{i,k}`

`(B4) Var[I_{i,k}] = barOmega_{i,k}^2 / m_i`

### 3.2 总干扰

`(B5) I_k = sum_{i=1}^{N_I} I_{i,k}`

若用 Gamma 分布匹配一阶和二阶矩，则

`(B6) E[I_k] = Omega_{Sigma,k} = sum_i barOmega_{i,k}`

`(B7) Var[I_k] = sigma_{I,k}^2 = sum_i barOmega_{i,k}^2 / m_i`

令：

`(B8) I_k approx Gamma(m_{eq,k}, theta_{eq,k})`

则必须满足：

`(B9) m_{eq,k} * theta_{eq,k} = Omega_{Sigma,k}`

`(B10) m_{eq,k} * theta_{eq,k}^2 = sigma_{I,k}^2`

解得：

`(B11) m_{eq,k} = Omega_{Sigma,k}^2 / sigma_{I,k}^2`

`(B12) theta_{eq,k} = sigma_{I,k}^2 / Omega_{Sigma,k}`

### 3.3 可直接写进正文的解释

这组公式的作用是：

- 不必把每个干扰源都写成复杂的逐点积分对象
- 仍然保留异构源的强弱和衰落差异
- 让后续 outage 推导可以直接走 `MGF` 路线

## 4. 闭式 outage 的主推导

这是最适合“升格”为论文命题的部分。

### 4.1 设定

令：

`(C1) S_{s,k} = barS_{s,k} * H_{s,k},  H_{s,k} ~ Exp(1)`

`(C2) I_k approx Gamma(m_{eq,k}, theta_{eq,k})`

`(C3) gamma_{s,k} = S_{s,k} / (I_k + N_k)`

### 4.2 条件 outage

给定 `I_k = i`：

`(C4) P(gamma_{s,k} < tau | I_k = i) = P(H_{s,k} < tau * (i + N_k) / barS_{s,k})`

由于 `H_{s,k} ~ Exp(1)`：

`(C5) P(gamma_{s,k} < tau | I_k = i) = 1 - exp(-tau * (i + N_k) / barS_{s,k})`

### 4.3 对干扰取期望

`(C6) P_out^{s,k}(tau) = 1 - exp(-tau * N_k / barS_{s,k}) * E[exp(-tau * I_k / barS_{s,k})]`

Gamma 随机变量的 `MGF/Laplace` 项为：

`(C7) E[e^{-t I_k}] = (1 + theta_{eq,k} * t)^(-m_{eq,k})`

令 `t = tau / barS_{s,k}`，得到：

`(C8) P_out^{s,k}(tau) = 1 - exp(-tau * N_k / barS_{s,k}) * (1 + theta_{eq,k} * tau / barS_{s,k})^(-m_{eq,k})`

### 4.4 对整网平均

`(C9) barP_out(tau) = 1 / (K * S) * sum_{k=1}^K sum_{s=1}^S P_out^{s,k}(tau)`

### 4.5 两个自然推论

#### Corollary 1

当 `theta_{eq,k}` 或 `Omega_{Sigma,k}` 增大时，`P_out^{s,k}` 单调增大。

#### Corollary 2

当 `m_{eq,k} -> +inf` 时，聚合干扰逐渐退化为确定性均值，此时

`(C10) P_out^{s,k}(tau) -> 1 - exp(-tau * (Omega_{Sigma,k} + N_k) / barS_{s,k})`

这条推论既能体现数学性，也和代码里的 `nakagami_sample_mode = mean` 有直接呼应。

## 5. 平均速率、Throughput 与 BLER 拟合

### 5.1 平均速率半闭式

如果不只想给 realized rate，而想给更像论文的表达，可写：

`(D1) barR_{s,k} = E[B log2(1 + gamma_{s,k})]`

利用积分恒等式：

`(D2) barR_{s,k} = B / ln(2) * int_0^inf (1 - P_out^{s,k}(t)) / (1 + t) dt`

这是一条非常标准、也很 AEU 的“半闭式”写法。

### 5.2 BLER 的工程化数学重写

当前代码真实来源是 `BLER_A.csv` 和 LUT 插值，因此最诚实的写法是：

`(D3) BLER(gamma_dB) = I_LUT(gamma_dB)`

但如果要增强数学美感，可以在论文里说明：

`(D4) I_LUT(gamma_dB) approx 1 / (1 + exp(a_b * (gamma_dB - b_b)))`

其中 `a_b, b_b` 由 `BLER_A.csv` 拟合得到。

于是：

`(D5) T_{s,k} = R_{s,k} * (1 - BLER(gamma_{s,k}))`

如果要写平均 throughput：

`(D6) barT_{s,k} = E[R_{s,k} * (1 - BLER(gamma_{s,k}))]`

## 6. 旋翼能效与速度劣化推导

### 6.1 推进功率

`(E1) P_prop(v) = P_0 * (1 + 3v^2 / U_tip^2) + P_i * sqrt(sqrt(1 + v^4 / (4v_0^4)) - v^2 / (2v_0^2)) + 0.5 * d_0 * rho * s * A * v^3`

### 6.2 通信能效

`(E2) EE = sum_{k,s} R_{s,k} / (sum_k P_prop(v_k) + P_tx,total)`

### 6.3 速度劣化映射

代码里非常适合写成：

`(E3) D_speed(v) = 1 - P_prop(0) / P_prop(v)`

这条式子很适合解释：

- 为什么 worst-case scene 不只是“干扰更大”
- 还可能是“高度/速度配置让单位能量可用吞吐更差”

## 7. Measured field 验证公式

### 7.1 IDW 场重建

`(F1) haty(x) = sum_n w_n(x) y_n / sum_n w_n(x)`

`(F2) w_n(x) = 1 / max(||x - p_n||, 1)^p`

### 7.2 poor-region proximity

`(F3) R_poor(x) = exp(-d_poor(x) / sigma_c)`

`(F4) C(x) = exp(-d_meas(x) / sigma_c)`

### 7.3 最终验证分数

`(F5) Score_field(x) = C(x) * (0.85 * R_metric(x) + 0.15 * R_poor(x))`

`(F6) hit_rate_delta = 1 / N * sum_n 1{d_poor(x_n) <= delta}`

### 7.4 这一节怎么写得像论文

重点不在“我们把数据画成图”，而在：

- 我们构造了一个 measured communication field
- 并把高风险区域发现写成了一个 field-driven search problem
