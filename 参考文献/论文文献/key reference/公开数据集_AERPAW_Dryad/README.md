## 数据集语义提醒

- `AERPAW` 和 `Dryad` 提供的是测量轨迹 / 测量点及其通信指标
- 它们不直接提供完整的真实：
  - 基站地图
  - 干扰源地图
  - 建筑物附着干扰源真值

因此，后续写作时应优先将其描述为：

- 公开测量数据
- 公开测量通信场 / 风险场
- 实测对照数据源

而不应直接表述为：

- 完整真实基站-干扰源标注地图
